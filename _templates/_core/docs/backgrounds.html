O Bootstrap, através de variáveis SASS, disponibiliza uma lista de cores contextuais (primary, success, danger...). Desde a versão 4 foram adicionadas variáveis referêntes às cores básicas como green, yellow, blue, entre outras. Por sua vez, a BASE adiciona mais algumas cores, visando uma paleta de cores quase completa para qualquer projeto.

Backgrounds:
Para permitir a utilização dessas cores como background, a BASE disponibiliza diversas classes customizadas.
Clique na cor para visualizar a variável e a classe referênte ao background.
