O Bootstrap disponibiliza uma paleta com as principais cores. Assim como disponibiliza classes para utilização direta dessas cores em backgrounds, bordas, texto e etc... O Template BASE adiciona ainda mais classes com o objetivo de oferecer mais recursos para customização de elementos. Confira abaixo todas as cores/classes disponíveis para a customização de elementos.
